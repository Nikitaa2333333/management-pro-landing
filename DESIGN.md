# Design System — They People (источник правды по стилям)

Минималистичный B2B SaaS: монохром, высокий контраст, много воздуха, границы вместо теней.

## 1. Цвета

### Основные
| Роль | HEX |
|------|-----|
| Primary / CTA / текст | `#000000` |
| Фон / поверхность / текст на тёмном | `#FFFFFF` |

### Нейтральные
| Токен | HEX | Назначение |
|-------|-----|-----------|
| Neutral 80 | `#A6A6A6` | вторичный текст, плейсхолдеры, подсказки |
| Neutral 90 | `#E6E6E6` | стандартные границы |
| Neutral 95 | `#F2F2F2` | лёгкие границы, разделители |
| Neutral 97 | `#FAFAFA` | hover-подложки, тонкая дифференциация |
| Neutral 100 | `#FFFFFF` | фон, карточки |

### Состояния
- Ошибка: border `#DC2626`, фон `#FEF2F2`
- Disabled: фон `#D0D0D0` / `#F2F2F2`, текст `#808080` / `#A6A6A6`

## 2. Типографика

Шрифт: **SF Pro Display**, fallback `system-ui, -apple-system, sans-serif`.
Веса: только **400** и **500**. Bold 600+ — запрещён. Акцент делать весом 500, не цветом.

| Роль | Size | Weight | Line-height |
|------|------|--------|-------------|
| Display 1 (Hero H1) | 40px | 500 | 44px |
| Heading 2 | 30px | 500 | 34.5px |
| Heading 3 (карточки) | 16px | 400 | 22.4px |
| Heading 4 (emphasis) | 16px | 500 | 22.4px |
| Body | 16px | 400 | 22.4px |
| Body small | 14px | 400 | 19.6px |
| Caption | 12px | 400 | 16.8px |
| Button | 16px | 400 | 22.4px |
| Link | 16px | 500 | 22.4px (underline) |

Letter-spacing везде `0`. Базовые размеры: **40 / 30 / 16px**. Line-height ≈ 1.4×.
Мобайл: H1 40→28px, body остаётся 16px.

## 3. Компоненты

### Кнопки (высота 46px, radius 6px, padding 12px 24px)
- **Primary**: фон `#000`, текст `#fff`, без бордера. Hover `#1A1A1A`, active `#0D0D0D`, disabled фон `#D0D0D0`/текст `#808080`.
- **Secondary**: фон `#fff`, текст `#000`, border `1px #F2F2F2`. Hover фон `#FAFAFA` + border `#E6E6E6`.
- **Ghost/Icon**: прозрачный, текст `#000`. Hover подложка `#F2F2F2`. Min touch 44×44px.

### Карточки
- Border `1px solid #F2F2F2`, radius `0` (острые) или `6px`. Без тени.
- Padding `24px`. Hover: border `#E6E6E6`, фон `#FAFAFA`.

### Инпуты / textarea (height 46px, radius 6px, padding 12px 16px)
- Фон `#fff`, border `1px #F2F2F2`, текст `#000`, placeholder `#A6A6A6`.
- Focus: border `#000` + `inset 0 0 0 1px #000`. Textarea min-height 120px, resize vertical.

### Навигация
- Прозрачный фон. Ссылки `#000`, size 16, weight 400/500. Hover текст `#606060` / opacity 0.7. Active — weight 500 / underline.

### Ссылки
- Inline: `#000`, weight 500, underline. Hover `#606060`.
- Footer: `#000`, weight 400. Hover opacity 0.7 + underline.

## 4. Layout и отступы

Базовая единица **4px**. Шкала: `4, 8, 12, 16, 24, 32, 48, 72, 84, 96, 168`.
- Внутри компонентов: 4–12px. Падинги карточек/инпутов: 16–24px.
- Между секциями: 48–96px (десктоп), на мобайле ÷2.
- Большие границы секций: до 168px.

### Сетка
- Max-width контейнера **1200px**, центрирован.
- 12-колоночная гибкая сетка. Gutter 32px (десктоп) / 16px (мобайл).
- Падинг контейнера: 48px десктоп → 24px планшет → 16px мобайл.

### Радиусы
`0` (карточки/острые), `6px` (кнопки/инпуты/модалки), `12px` (мягкие карточки, опц.), `50%` (аватары).

## 5. Тени (минимум — только на hover)
| Уровень | Тень |
|---------|------|
| Base | `none` |
| Hover | `0 2px 8px rgba(0,0,0,.08)` |
| Overlay (модалки) | `0 4px 16px rgba(0,0,0,.12)` |
| Deep | `0 8px 32px rgba(0,0,0,.16)` |

Глубина создаётся **границами и воздухом**, а не тенями.

## 6. Брейкпоинты
| Имя | Ширина | Ключевое |
|-----|--------|----------|
| Mobile | 320–639 | 1 колонка, padding 16, H1 28px, кнопки на всю ширину |
| Tablet | 640–1023 | 2 колонки, gutter 24, padding 32 |
| Desktop | 1024–1199 | 3–4 колонки, gutter 32, padding 48 |
| Large | 1200+ | контейнер 1200px, секции до 168px |

Touch-таргеты ≥ 46×46px. Между интерактивными элементами ≥ 8px по вертикали / 12px по горизонтали.

## 7. Do / Don't
**Do:** высокий контраст (чёрное на белом), много воздуха, только SF Pro Display, чёрный для ключевых элементов, radius 6px на интерактиве, границы `#F2F2F2` вместо теней, сетка 4px, акцент весом 500.
**Don't:** декоративные тени, смешивать шрифты, bold 600+, тесные лейауты (<12px вокруг элемента), кастомные серые вне шкалы, цветные бордеры на кнопках, радиусы >6px (кроме мягких карточек/аватаров), избыточный декор.
