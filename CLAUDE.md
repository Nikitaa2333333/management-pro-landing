# МЕНЕДЖМЕНТ.PRO — Landing

Одностраничный лендинг (single-page). Цель — **повторить дизайн** референса «They People»,
но с текстами и брендом **МЕНЕДЖМЕНТ.PRO** (система привлечения B2B-клиентов через холодный аутрич).
Главное требование: максимально простой сайт, **открывается мгновенно** (статика, минимум JS).

## Стек
- **Astro** — статический генератор. На выходе чистый HTML, near-zero JS → мгновенная загрузка.
- **Tailwind CSS** — стили утилитами. Дизайн-токены вынесены в `tailwind.config`.
- Шрифт: **SF Pro Display** (fallback: `system-ui, -apple-system, sans-serif`).

## Источники правды
- **Дизайн** → [DESIGN.md](DESIGN.md). Строгий монохром (чёрный/белый/серые), без акцентных цветов.
- **Тексты** → [content.json](content.json). Вытянуты скриптом **дословно** из
  `Тексты_лендинга_Холодный_аутрич_МЕНЕДЖМЕНТ_PRO.docx` (колонки «ПЕРЕПИСАНО»).
  ⚠️ Тексты НЕ переписывать вручную. При изменении исходника — перегонять скриптом, не править руками.

## Принципы
- Веса шрифта только **400 и 500** (никакого bold 600+).
- Контейнер max-width **1200px**, много воздуха, границы вместо теней.
- Русскоязычный сайт → применять типографику (неразрывные пробелы, висячие предлоги) через скилл `russian-typographer`.

## Структура страницы (ключи из content.json)
1. **Хедер** — лого `brand` + `nav` (Как работает · Кейсы · Методика · Блог · Контакты)
2. **Hero** — `hero` (label, h1, subtitle, cta_primary, cta_secondary)
3. **Solution** — `solution` (label «КАК ЭТО РАБОТАЕТ», text[])
4. **3 карточки ценности** — `value_cards[]` (title, desc)
5. **Features / Подход** — `features` (label, headline, text, cta, bullets[])
6. **Trust** — `trust` (heading, logos[], stat)
7. **Отзывы** — `testimonials[]` (badge, quote, author)
8. **Медиа / Блог** — `media` (label, h2, subtitle, cta, cards[4])
9. **FAQ** — `faq[]` (q, a) — аккордеон, 5 вопросов
10. **Финальный CTA** — `final_cta` (label, headline, text, cta, note)
11. **Футер** — `footer` (line1, line2, line3) + контакты @nikolay_levn / +7 966 076 6605
